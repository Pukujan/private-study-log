## Immediate blockers to a truthful governance claim

1. Restart/replay the Hades gateway and prove the corrected absolute MCP launcher, canonical write
   workspace, and assured status on the live client path.
2. Produce a fresh availability probe, signed exact-model capability cards, objective task
   scorecards, and a generic route-to-`delegate_task` actuator whose actual provider call is externally
   observable. The current route correctly returns `UNRESOLVED`.
3. Install an external assurance trust root and separate preflight/research/evaluator/human signer
   services. Private keys must not enter Hades or the builder workspace.
4. Restart Hades and replay the installed standalone assured-driver hook, proving it blocks legacy
   starts and raw delegation until the signed preflight, route binding, assured run, and joined IDs exist.
5. Query gravebuster OTel/Langfuse with the same run/task/route IDs and run the hidden CaseOS evaluator
   before beginning any real product-quality claim.
